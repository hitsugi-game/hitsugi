# ビジュアル復旧・出立・ダンジョン刷新計画

**策定日**: 2026-07-11  
**実装担当**: Claude / Fable 5  
**素材担当**: Codex生成候補＋ComfyUI工場  
**前提**: ゲームソース実装は本書を受けたClaudeが行う。

## 1. デザインの主題

対象は、世代交代RPGを好み、数値の攻略だけでなく「一族が常夜を歩いた記憶」を残したいプレイヤー。

出立画面の一つの仕事は、**次の土地を恐れながら選びたくさせること**。ダンジョン画面の仕事は、**同じ一歩を、土地固有の発見へ変えること**。

採用する美術方向は「夜藪の地層」。絵巻を平面の路線図として扱わず、一族が灯で暴いた土地の記憶が、和紙の層として浮かぶ地図にする。

## 2. トークン

既存「墨金の家譜」を壊さず、探索専用の役割色を追加する。

| 役割 | 名称 | 色 |
|---|---|---|
| 最深背景 | 常夜墨 | `#070A14` |
| 地図地 | 夜藪藍 | `#111A33` |
| 行ける道/灯 | 道灯 | `#E6A34A` |
| 未知/霧 | 月灰 | `#A9B7D8` |
| 主/不可逆危険 | 傷朱 | `#A9483D` |
| 既知/鎮 | 灰金 | `#8D887D` |

書体:

- 地名・異名・章題: `Shippori Mincho`。字間を広げ、短く使う。
- 操作・数値・危険情報: `BIZ UDPGothic`, `Yu Gothic UI`, sans-serif。明瞭性を優先。
- 家譜引用と伝承: 明朝、行間1.8。

既存画面の全てをゴシックへ変えない。地名の詩情と、判断情報の読みやすさを役割で分ける。

## 3. 署名要素「灯写し」

ノードを選ぶと、右の地域画が暗い和紙から灯に透かされるように一度だけ現れる。

- 0〜120ms: 暗い和紙と地形輪郭。
- 120〜350ms: 中央から灯色が広がり地域画を見せる。
- 350ms以降: 静止。常時脈動しない。
- 主が未討伐なら、地域画の奥にボスの輪郭だけ残る。
- 討伐後は輪郭が消え、家の「鎮」印が押される。
- reduced motionではクロスフェードのみ。

これを本画面唯一の大きな演出とし、粒子・グロー・ホバー浮遊は抑える。

### 4.3 初期状態

- 開いた時点で最前線の解禁地域を自動選択する。
- 前回選んだ地域が現在も有効ならそれを優先する。
- 右側の空状態は禁止。
- 画像404時は同じ森林背景へ黙って差し替えず、地域名入りの墨絵シルエット＋「遠見が利かぬ」を表示する。

### 4.4 情報開示

常時表示:

- 地名、tier、深さ、主の有無、討伐状態。
- 想定される地形1語。「湿地」「古道」「断崖」「館」など。
- 主の異名と輪郭。

物見櫓で追加:

- Lv1: 敵属性傾向。
- Lv2: 危険床・灯消費傾向。
- Lv3: 主の危険な一手と推奨耐性。

攻略答えを全開示せず、準備に必要な情報だけを段階化する。

## 5. 地域画とボス画の使い分け

- 地域背景: 行きたいと思わせる「場所」。人物を主役にしない。
- ボスの間: 戦う場所。中央下を空け、戦闘キャラを重ねられる構図。
- ボス立ち絵: 背景なしに近い暗地、輪郭と手足が読める。
- カットイン: 初遭遇の0.8〜1.2秒。横長、目または象徴物を焦点にする。

出立ではボス立ち絵を全面公開しない。立ち絵を黒い輪郭で使い、初遭遇で色と全身を解禁する。

## 6. ダンジョン刷新「四幕の道行き」

全フロアを同じ密度にしない。各地域を四幕で進める。

### 第一幕 — 閾

- 入場時に地域背景を2秒以内の短い導入として見せる。
- 地名、環境音、署名ランドマークの遠景。
- 操作は止めず、移動入力で即消える。

### 第二幕 — 痕

- ボスを示す痕跡を一つ置く。例: 蛍の群れが全て同じ方向を向く。
- 調べると主の挙動ヒントまたは家譜断片。
- 見逃しても進行不能にしない。

### 第三幕 — 畏

- 最終前フロアで色・音・粒子を変える。
- 通常プロップを減らし、主の署名物を増やす。
- 主の一手を示す痕跡をもう一つ置く。

### 第四幕 — 座

- ボスの間背景へ切り替える。
- 参道、門、無音の間、カットイン、戦闘の順。
- 勝利後は同じ場所の光・粒子だけ変え、鎮まったことを見せる。

## 8. ダンジョン内の遊びへの接続

装飾を攻略情報にする。

- 痕跡を1つ見る: ボス属性を開示。
- 2つ見る: 危険技の予兆を開示。
- 3つ見る: 初回だけ戦闘開始時に「見切り」小効果。または回避方法を明示。
- ランドマークを家譜へ写す: 地域短話と地図の印が増える。
- 討伐後の再訪: 敵密度だけでなく光・住民の痕跡・採取物が変わる。

単なる収集物を増やさず、ボス準備と物語を同じ観察行動で進める。

## 9. 敵影

現在の手続き生成の敵影は性能面で良いが、地域差が弱い。

- 基礎種画像から3色まで抽出した輪郭影を事前生成またはロード時生成する。
- 遠距離は輪郭と目だけ、接近で種族特徴を一つ見せる。
- 若/老は大きさだけでなく姿勢と歩行周期を変える。
- 金個体は全面金色にせず、部位一つと足跡だけ金にする。
- ボス階では雑魚影を減らし、主の痕跡を優先する。

## 11. 自己批評

最初の案は「暗色＋金＋カード」の延長で、別の和風RPGにも流用できる構成だった。そこで次を変更した。

- カード一覧を主役にせず、和紙の地層と灯写しを主役にした。
- 40ノードを同じ強さで見せず、現在地周辺を地理として読ませる。
- 背景追加だけで終えず、痕跡をボス攻略へ接続した。
- 常時粒子やグローを増やさず、選択時の一度だけの灯写しへ演出予算を集中した。
