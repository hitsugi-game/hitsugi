# 『灯継ぎ』 -HITSUGI- ゲームデザインドキュメント

**ジャンル**: 世代交代ダークファンタジーRPG(ブラウザ/短セッション設計)
**プレイ時間**: 1周 2〜4時間(1セッション5〜15分で区切れる)
**プラットフォーム**: Web(PC/スマホ対応)
**インスパイア**: 『俺の屍を越えてゆけ』のシステム的遺伝子(世代交代・短命・血脈)を受け継ぎつつ、物語・世界観・キャラクターは完全オリジナル。現代的UXで再設計。

---

## 0. 「人が楽しむ」とは何か — 本作の設計哲学

俺屍が四半世紀愛される理由は「悲しさが楽しさに変換される」構造にある。本作はこれを分解し、5つの快楽原理として実装する。

1. **喪失による愛着** — キャラは必ず死ぬ。だからこそ1シーズンが愛おしい。「もう会えない」が「忘れない」に変わる装置(家譜・形見・遺言)を用意する。
2. **自分だけの物語の自動生成** — プレイヤーごとに家系図・名前・死に様が違う。システムが物語を「書かせる」のではなく「書き残す」。スクリーンショット1枚で人に語りたくなる。
3. **意思決定の重み** — 「今季、誰と契るか」「灯が尽きる前に引き返すか」。全ての選択が世代を超えて響く。ただし**理不尽ではなく透明**に(俺屍の不透明さは改善点)。
4. **プッシュ・ユア・ラック** — 探索は「灯」が尽きる前に帰るか、深部の宝を狙うかの賭け。緊張と緩和のリズムを5分単位で作る。
5. **上達と発見** — 血脈の掛け合わせ・技の継承ルートの発見は攻略の喜び。「次の世代ならもっとうまくやれる」が再挑戦の動機になる。

**アンチゴール**(やらないこと): 作業的な周回強制 / 不透明な内部計算 / 長すぎるチュートリアル / ガチャ的搾取感。

## 1. 世界観・ストーリー

### 1.1 舞台 — 燈ノ郷(あかしのさと)

山あいの盆地にある小さな郷。千年前から**常夜(とこよ)** — 太陽の昇らない永い夜 — に呑まれており、郷は中央の**大燈籠(おおとうろう)**の火だけで闇と獣を退けて生きている。大燈籠の火は年々細り、郷の外は**夜藪(よやぶ)**と呼ばれる魔性の領域が広がる。

### 1.2 発端 — 二つの呪い

千年前、旅の楽士**汐里(しおり)**は、常夜をもたらす**玄冬(げんとう)**という星喰いの神から郷を守るため、自らの寿命を薪として大燈籠に火を点した。だが火は完成せず、汐里の血族には二つの呪いが残った。

- **八季の命** — 一族は生まれてから八つの季節(二年)で、体の内から灯が燃え尽きて死ぬ。
- **結ばれぬ身** — 一族は人と子を成せない。

一族が絶えれば大燈籠は消え、郷は常夜に沈む。ゆえに一族は夜空に残った**星神(ほしがみ)**たちと**星契り(ほしちぎり)**を結び、半神の子を産み、代を継いで戦い続けてきた。

### 1.3 目的 — 灯の御山へ

夜藪の最深部、**灯ノ御山(あかしのみやま)**の頂に玄冬が座す。歴代の当主は誰一人頂に届かなかった。プレイヤーは汐里の末裔の一族を率い、世代を重ねて御山の頂を目指す。

### 1.4 真相(終盤の転回)

玄冬の正体は**汐里そのひと**である。千年前、汐里は玄冬を滅ぼせず、己の身に封じて山頂に座した。星喰いは汐里の体を蝕み、いま彼女自身が常夜の源になりかけている。八季の命の呪いは、実は汐里が末裔に遺した**保険** — 一族の命を薪として大燈籠に送り、自分が完全に星喰いに堕ちるのを千年遅らせるための仕掛けだった。

最終戦は「先祖殺し」ではなく**「先祖の看取り」**。倒すのではなく、千年戦い続けた家祖を眠らせ、その灯を継ぐ。エンディングで一族の呪いは解け、最後の当主は「八季で死なない初めての子」を抱く。タイトル『灯継ぎ』(ひつぎ=柩)の意味がここで回収される。

### 1.5 トーン

和風ダークファンタジー。悲壮だが湿っぽくしない。郷の人々は明るく、星神たちは可笑しみがあり、死は「祭」として送られる。**切なさ7:温かさ3**。

## 2. 登場キャラクター

### 2.1 導き手

- **綴(つづり)** — 家譜を千年書き続ける付喪神(筆の精)。ナビゲーター兼ツッコミ役。口が悪いが一族を誰より愛している。死んだ当主全員の名を諳んじられる。「……また一人、書かねばならんのか。癖でな、涙は墨に変えることにしとる」
- **汐里(しおり)** — 家祖にして最終ボス。回想と「夢渡り」イベントで断片的に登場。楽士らしく、戦闘BGMのモチーフは彼女の子守唄の変奏。

### 2.3 一族(プレイヤーキャラ)

自動生成される名前(和名プール)+血脈による外見・声質の変化。各キャラは以下を持つ:

- **血潮(けっちお)**: 力・体・技・速・心・運の6系統ポテンシャル(親と星神から遺伝)
- **性根(しょうね)**: 「勇敢」「臆病」「慈悲深い」「負けず嫌い」等 → 戦闘AI補正・台詞・遺言が変わる
- **遺言・辞世**: 死亡時に性根と生涯実績から自動生成され家譜に残る

## 3. ゲームシステム

### 3.1 時間 — 季節ターン制

1ターン=1季節(春夏秋冬)。一族は生後8季で必ず死ぬ。毎季、以下から行動を選ぶ:

1. **出立(しゅったつ)** — 夜藪へ探索(メインアクション、後述)
2. **星契り** — 当主候補が星神と契り、子を授かる(翌季誕生、2季で成人)
3. **祭** — 郷で祭を開く: 一族全員の士気UP+親密度UP(回数制限あり)
4. **静養** — 傷と心労を癒す

**UX改善**: 俺屍の「月単位で細かすぎる」問題を季節単位に圧縮。1ターン=1意思決定=約5分。

### 3.2 星契り(世代継承)

- 子は親の血潮×星神の血潮から**予測レンジ表示付き**で生成(俺屍の不透明さを改善)
- 技(奥義)は「継承の証」として最大2つ持ち越し可能
- 形見(装備)は世代を超えて受け継がれ、継承回数で強化される(「祖母の簪+3」)
- 奉燈(通貨)を多く捧げるほど高位の星神と契れる

### 3.3 探索 — 夜藪行(よやぶゆき)

ノード式マップ(分岐路を選んで進む)。各ノード: 戦闘/宝/焚火(回復)/事件(イベント)/枝道。

**灯(ともしび)システム** — 中核のプッシュ・ユア・ラック:
- 出立時に灯100。ノード移動・戦闘で減少。
- 灯が尽きると「常夜濃度」が上がり敵が凶悪化、被ダメ増。全滅=持ち帰り半減+死者は「夜藪に消える」(家譜に「行方知れず」と記される)。
- 「帰り火」を使えばいつでも帰還可能(成果は確定持ち帰り)。**深く行くほど報酬は跳ね上がる**。

### 3.4 戦闘 — 円陣式コマンドバトル

- 最大4人パーティ、ターン制コマンド。**前列/後列**の陣形。
- 属性: 火・水・風・土・月・星の6属(相克関係を画面内に常時ヒント表示)
- **継足(つぎたし)** — 家族の絆メカニクス: 同じ敵を家族が連続で狙うと連携ダメージ倍率UP。親子・兄弟の組み合わせで専用連携台詞。
- 敵を倒すと**奉燈**(通貨)と**血珠**(強化素材)をドロップ。
- 戦闘は1戦90秒以内を目標にテンポ重視。倍速・オートあり。

### 3.5 家譜(かふ) — 自動生成される物語

全キャラの生涯が自動記録される巻物UI: 名・親・契った神・討った敵・性根エピソード・辞世。
**共有機能**: 家譜を画像出力してSNS共有(マーケティングの核)。

### 3.6 経済・強化

- **奉燈**: 星契り・装備購入に使用
- **血珠**: 血潮の底上げ(控えめ。基本は遺伝で強くする)
- 郷の施設(鍛冶屋・薬師・社)が世代進行で発展

### 3.7 難易度・セッション設計

- オートセーブ(localStorage)。いつ閉じても再開可能。
- 「語り部モード」(戦闘易化)と「宿命モード」(標準)。
- 1周クリア目標: 12〜20世代、2〜4時間。クリア後は「継承新周回」。

## 4. アートディレクション

- **画風**: 和風水彩+切り絵。墨の輪郭、にじむ暖色の灯、藍の夜。
- **UI**: 和紙テクスチャ、縦書きアクセント、金泥の罫線。フォント: 明朝系(Shippori Mincho)。
- **必要アセット**(codex生成): タイトル絵、星神12柱の立ち絵、敵15種、背景6種(郷・夜藪各層・御山)、UI装飾。
- **音楽**: Web Audio による和風プロシージャル+作曲シーケンス(汐里の子守唄モチーフを全曲に織り込む)。

```
タイトル → (初回)導入の語り → 郷(ホーム)
郷 ─ 出立 → ノードマップ → 戦闘/事件 → 帰還清算 → 季節経過
   ─ 星契り → 星神選び(子の予測表示) → 誕生演出
   ─ 家譜 / 祭 / 静養 / 鍛冶
死亡時: 看取り演出 → 辞世 → 家譜に記帳
```
