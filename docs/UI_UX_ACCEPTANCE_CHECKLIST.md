# UI/UX刷新 受入チェックリスト

> 対応設計: `docs/UI_UX_REDESIGN_PLAN.md`
> 用途: Claude実装時の自己確認、レビュー、出荷前確認

## A. 共通殻

- [ ] 独立作業画面は左上に戻る、画面題、必要な資源を持つ
- [ ] 長い作業画面の主要CTAは下部固定
- [ ] モーダルは右上に閉じるを持つ
- [ ] ESCでモーダルを閉じられる
- [ ] モーダルを閉じると呼び出し元へフォーカスが戻る
- [ ] モーダル中のTabフォーカスが背後へ抜けない
- [ ] 背景スクロールがロックされる
- [ ] クリック可能な非button要素が残っていない、またはrole/keyboardを備える
- [ ] PC 1280×720で主要CTAが画面外へ消えない
- [ ] モバイル390×844で意図しない横スクロールがない

## B. 郷ホーム

- [ ] 1〜2人時は一族欄が「人物札＋血脈診断」になる
- [ ] 3人以上で人物選択により大札が切り替わる
- [ ] HP30%未満は「瀕死/傷深し」などの状態語が出る
- [ ] 後継なし、成人なし、懐妊中が一目で分かる
- [ ] 状況に応じて推奨行動が切り替わる
- [ ] 推奨理由が1行で表示される
- [ ] 非推奨行動も選べる
- [ ] 祭/静養は次月・回復・残寿命を確認してから実行する
- [ ] 無効カードの理由が通常コントラストで読める
- [ ] 郷の帳の各入口に現在値または新着が表示される
- [ ] 新着/達成が0のbadgeは表示しない
- [ ] モバイルのホーム内アンカーが各節へ正しく移動する

### 危機状態テスト

- [ ] 存命1・後継なし・契り可能 → 星契り推奨
- [ ] 全成人HP30%未満・後継あり → 静養推奨
- [ ] 成人0・幼子あり → 成人までの残月を表示
- [ ] 祭月・複数負傷 → 祭推奨
- [ ] 奉燈不足 → 不足量と出立導線を表示

## C. 星契り

- [ ] 親/星/見立ての現在ステップが分かる
- [ ] 選択中の親と星神が固定領域で確認できる
- [ ] 奉燈費用と残高がCTA付近で確認できる
- [ ] 得意能力上位2つを一覧で比較できる
- [ ] 子の見立てを数値範囲と短文の両方で理解できる
- [ ] ロック理由が読める
- [ ] モバイルで一覧・詳細・見立てが長い一枚画面にならない
- [ ] 最終CTAがスクロールしても見失われない

## D. 出立/探索

- [ ] 行き先・隊人数・灯の準備状況が固定表示される
- [ ] 地域の危険度、深さ、主、未回収が比較できる
- [ ] 隊員の現在HP、残寿命、前後列が見える
- [ ] 出立CTAに地域名、人数、月消費が出る
- [ ] 加護選択で横スクロールが発生しない
- [ ] 探索中央をHUDが塞がない
- [ ] 帰り火確認に現在の持帰り量が出る
- [ ] 敵接近を視覚・音・ミニマップで予告する
- [ ] モバイルD-padがキャンバス/ログと重ならない

## E. 戦闘

- [ ] 戦場が画面高の60%以上ある
- [ ] 1人/1体でも敵味方が小さく隅へ寄らない
- [ ] 現在手番と次手が一画面で見える
- [ ] 継足対象、連数、次撃倍率が見える
- [ ] 継足対象の敵札にも印がある
- [ ] ログは直近2〜3行で、全履歴を展開できる
- [ ] 1人の隊員札が横いっぱいに伸びない
- [ ] 通常攻撃を1操作で実行できる
- [ ] 技一覧の戻るが常に見える
- [ ] 対象選択中に指示文が表示される
- [ ] 対象をキーボードで選べる
- [ ] オート設定が戦闘を跨いで維持される
- [ ] オート勝利後に自動復帰する
- [ ] reduced motionでもダメージ/会心/勝敗情報が残る

## F. 鍛冶と蔵/普請

- [ ] 鍛冶と蔵が購う/装備/打ち直し/鍛錬の4タブに分かれる
- [ ] 選択人物が固定表示される
- [ ] 装備前後の差分が増減記号付きで見える
- [ ] 新着/装備可能/未装備で絞り込める
- [ ] 購入後の残高がトーストで分かる
- [ ] 高額打ち直しは費用と予測を確認する
- [ ] 装備810件を一度にDOMへ描画しない
- [ ] 普請カードに現在Lv、次効果、費用が固定位置で出る
- [ ] 普請後に画面が閉じない

## G. 記録画面

- [ ] 家譜は概要/称号/故人/年代記に分かれる
- [ ] 図鑑タブが上部固定
- [ ] 図鑑に新着だけを見る手段がある
- [ ] 未知項目を大量の同型カードで埋めない
- [ ] 詳細文は選択後の詳細面で読む
- [ ] 家系図が全画面で開く
- [ ] 当主へ戻る、人物検索、存命絞り込みがある
- [ ] 共有/画像保存が「残す」操作としてまとまる

## H. 文章と状態

- [ ] 同じ操作名を入口/CTA/完了トーストで統一する
- [ ] 「閉じる」「郷へ戻る」「やめる」の用途が混ざらない
- [ ] エラーに原因と解決手段がある
- [ ] 空状態に次の行動がある
- [ ] 無効理由をtitle属性だけに依存しない
- [ ] 重要文字が12.5px未満でない
- [ ] 選択状態が色だけに依存しない

- [ ] console error 0

## J. 実機シナリオ

- [ ] 新規開始→出立→初戦闘→帰還
- [ ] 存命1/後継なし→星契り→誕生
- [ ] HP1の当主→静養確認→回復
- [ ] 鍛冶で購入→装備→差分確認
- [ ] 郷普請で不足→出立導線→帰還後に建設
- [ ] 図鑑新着→詳細→戻ると元位置を維持
- [ ] 家系図で当主→祖先→子へ移動
- [ ] ボス戦→戦果→探索復帰
- [ ] 全滅→生還/死亡/次画面遷移
